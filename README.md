# Art Therapy Plus

A comprehensive platform for art therapy, combining a marketing website and web application.

## Project Structure

```
ArtTherapy-Site/
├── frontend/          # Marketing website (React + Vite + TypeScript)
└── README.md          # This file
```

## Frontend (Marketing Site)

The frontend directory contains the Art Therapy Plus marketing website built with:
- React 19
- TypeScript
- Vite
- Tailwind CSS v4
- shadcn-ui components

See [frontend/ART_THERAPY_PLUS_README.md](./frontend/ART_THERAPY_PLUS_README.md) for detailed documentation.

### Quick Start

```bash
cd frontend
npm install
npm run dev
```

## Color Palette

- **Primary Blue**: `#2E3780` - Headers, buttons, primary elements
- **Secondary Pink**: `#F0BEBA` - CTAs, highlights, accents
- **Accent Orange**: `#FF7500` - Active states, secondary accents

## Pages

1. **Home** - Hero, problem statement, features
2. **How It Works** - 3-step process explanation
3. **About** - Mission, vision, team
4. **News & Media** - Filterable article listing
5. **Contact** - Contact form with mailto functionality
